About R/W Book Club

We've been meeting since April 2003. The "R/W" stands for Read / Write — we're a technically minded group, and many of us blog. We read, but we also write.

Each year we read about eight books together, mostly non-fiction. Members take turns picking the next book and hosting the meeting where we discuss it. This site is the running ledger of what we've read — scroll the reading journey to follow it back to the very first meeting.

How this site is built

Everything we've read lives in a database that the club's assistant, Oliver, keeps current — one record per book, meeting, member, and author. A small Python pipeline turns that into per-page content, fetches and resizes the cover art, and hands the result to Eleventy, which renders these pages. Oliver rebuilds and publishes the site to GitHub Pages whenever the club's record changes.
